О программе: SIR Мои финансы и бизнес
Программа проста в использовании и не требует
специальных знаний в области учета.
1. Для начала работы  нажать кнопку Настройи :	#1:
которая находится в правом верхнем углу
Затем выбрать пункт "Ваш идентификатор" и нажать
кнопку ⁠Изменить (находится в правом верхнем углу) 
В появивщейся области ввода заменить User на Ваше 
имя (идентификатор).
Внесённые изменения необходимо зафиксировать 
нажав кнопку ⁠Записать
Выход на предыдуший экран производится нажатием 
кноки находящися в левом верхнем углу
2. Для работы с данными необходимо нажать кнопку
Загрузить данные за период.
У Вас появится список Счетов, затем список
Контрагентов (субсчета), далее данные по 
контрагенту, которые можно добавлять (+), 
редактировать(Edit),
удалять(Del),
копировать(Copy)
